# AG-011 — Draw Instruction

## Commit

`AG-011: implement concrete Draw spell instruction`

## Implemented

- Optional numeric `Amount` on spell instruction definitions.
- Concrete `draw` instruction handler.
- Draws for the finalized spell's controller.
- Multi-card draws preserve deck order.
- If the deck runs out, already drawn cards remain drawn and the instruction
  succeeds after doing as much as possible.
- Draw and empty-deck events tied to the resolving Chain item.
- Full spell-resolution integration test.

## Deliberate boundary

This milestone implements only the concrete Draw instruction. It does not yet
implement draw-trigger dispatch, second-card-per-turn tracking, Predict, Reveal,
or Recycle.
