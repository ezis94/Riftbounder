# AG-005 — Generic Chain

## Commit

`AG-005: implement transactional Chain resolution`

## Scope

This milestone intentionally implements only the generic LIFO Chain container
and resolution coordinator. It does not implement priority, passing, Reactions,
card plays, triggered abilities, or the First Effect Priority Rule.

## Added

- `ChainItemId`
- `IChainItem`
- `Chain`
- `ChainResolutionResult`
- `IChainItemResolver`
- `ChainResolver`
- bulk-resolution result types
- transactional resolution semantics
- GitHub Actions build/test/format workflow
- ADR 0001
- 19 focused tests

## Rules relationship

The Chain is described in Core Rules 327–335. This milestone supplies the
generic storage and LIFO resolution primitive required by those rules.

Priority, passing, and the exact rules that decide when one Chain item resolves
are deferred to AG-006.

Canonical rules:
`https://www.riftbound.one/rules/riftbound-core-rules.pdf`
